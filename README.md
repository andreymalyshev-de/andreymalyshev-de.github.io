# Web-Portfolio

A website about myself - feel free to take a look: andreymalyshev.com


📜 The goal of creating this webpage was firstly to show my professional portfolio not only as a CV, but also as an interactive website. Secondly, I wanted to gain some experience in the JS/TS/CSS field and to understand how internet pages are built even the world of today where you can get your website done in 5 minutes by some AI. And to build this website by my own made actually a lot of fun :)

🛠️ For the design-tuning I used mainly CSS and for the interaction logic JavaScript.
 - The last and the main challenge I faced so far was implementing the dark theme toggling button. The logic and design was inspired by this repository: https://github.com/thecodingscape/light-dark-theme-toggle. 
 Here I e.g. began to use shared variables in CSS in order to provide unity of the color palette.
